## Relatório Analítico de Performance de E-commerce

Este repositório contém a análise exploratória e inferencial de dados de um e-commerce brasileiro, realizada pela equipe **Levi Moraes e Thays Barbosa**, com foco em KPIs de Receita, Logística e Comportamento do Cliente.

### Entregáveis Principais

*   **Relatório Analítico (PDF/MD):** Contém o Sumário Executivo, Análise de Dados & Método, EDA, Inferência Estatística e os Insights Acionáveis.
*   **Notebook Python (`notebooks/notebook_analise_ecommerce.ipynb`):** Código completo para reprodutibilidade da limpeza, feature engineering, EDA e Inferência.
*   **Script SQL (`code/sql/main_query.sql`):** Consulta SQL para reprodutibilidade da etapa de feature engineering em ambiente de banco de dados.
*   **Gráficos de EDA (`docs/`):** Visualizações geradas durante a análise.
